## Договоры (шаблон и генерация)

### Назначение
- В папке хранится шаблон договора для Telegram-бота.
- Сгенерированные договоры сохраняются в `generated/`.

### Файлы
- `dogovor-shablon-tg.docx` — шаблон договора.
- `generated/` — папка для готовых документов.

### Размещение на сервере
Папка должна быть доступна контейнерам `contract-bot` и `web` через volume.
